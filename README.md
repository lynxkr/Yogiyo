# Project 'Jogiyo'

## 요기요 카피 프로젝트

### [영상](https://www.youtube.com/watch?v=HKcluYaFeUY)

#### 개요

패스트캠퍼스 iOS 개발 스쿨 8기 수료 프로젝트로써, 요기요의 기능을 비롯한 모든 것을 따라한 프로젝트입니다.



```
주요 기능
1. 내 위치 기반 주변 음식점 목록 불러오기
2. 음식점 메뉴 보기, 리뷰 보기
3. 장바구니 담아두기 기능
4. 주문 기능
```
